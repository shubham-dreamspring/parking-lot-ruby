# Parking Lot

## Usage
### Help
```
./app.rb -h | --help
```
### Park Car
```
./app.rb -p | --park REGISTRATION_NUMBER
```
### Unpark Car
```
./app.rb -u | --unpark REGISTRATION_NUMBER
```
### Get all the cars in parking lot
```
./app.rb -c | --cars
```

### Get all the invoices
```
./app.rb -i | --invoice
```
### Get the invoice by invoice id
```
./app.rb -i | --invoice INVOICE_ID
```

### For testing
```
rspec
```
